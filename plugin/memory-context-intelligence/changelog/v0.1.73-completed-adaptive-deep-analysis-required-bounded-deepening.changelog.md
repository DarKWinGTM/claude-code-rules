## Summary

Closed the smallest missing adaptive deep-analysis gap for `memory-context-intelligence`: when the rendered analysis payload says bounded deepening is required, the analysis skill now has to perform one bounded deepening pass for the required top topic candidates before the first response, instead of leaving deepening as optional metadata only.

- kept the deepening step read-only, advisory-only, and trace-anchored
- preserved repeated topic-card output while allowing stronger mechanism / workflow / stop-gate explanation inside the cards
- added focused contract coverage for the stronger adaptive execution contract
