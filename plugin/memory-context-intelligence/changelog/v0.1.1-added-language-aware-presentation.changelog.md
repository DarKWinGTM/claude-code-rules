# Version 0.1.1 - Added language-aware presentation to the topic-list and choice flow

> **Parent Document:** [changelog.md](changelog.md)
> **Current Version:** 0.1.10
> **Session:** 20625882-e064-4b40-a415-8d425cf392dc (2026-05-16)

---

## Changes
- updated the capsule README so the intended flow now explicitly presents topic lists and deeper expansions in the user's active working language where practical
- updated the main concept design so user-facing explanations are language-aware while canonical technical identifiers remain exact
- updated the topic-list and choice-flow design so localization applies to headings, descriptions, summaries, and choice prompts
- added a boundary that identifiers, file paths, commands, config keys, env vars, and exact evidence anchors must not be translated loosely

## Summary
This version strengthens the presentation model so the capsule can explain memory-derived analysis in the user's native or active working language without weakening authority, evidence, or identifier fidelity.
