## Objective

Define when and how externally researched evidence should strengthen candidate improvement topics that were first detected from local work traces.

## Why this phase exists

Local work traces are strong for finding recurring pain and recurring lessons, but they are not always enough to justify a strong proposal. Some topics need external theory, standards, best practices, or broader guidance before they can become trustworthy candidate improvements.

## Action points

- [x] define when research is needed
- [x] define the single full-power evidence-backed method
- [x] define how local and external evidence are synthesized
- [x] define source-trust boundary
- [x] keep external research as proposal-strengthening input rather than authority override

## Verification

- research need is treated as topic-specific, not universal
- the single full-power evidence-backed method is explicit
- stronger proposals still remain candidate-only until selected and verified
- external research does not override current RULES authority or user instruction

## Exit criteria

- the capsule has a clear research-enrichment stage for topics that need broader support
- the concept now routes broader support through one full-power method instead of several user-facing research modes
